# VHS Horror Puzzle Collection - Product Overview

## Project Purpose
A horror-themed puzzle game collection featuring classic word games (Connections, Wordle, Strands, Spelling Bee, Letter Boxed) with an immersive VHS aesthetic and narrative storytelling elements.

## Key Features

### Core Gameplay
- **Multiple Puzzle Types**: Connections (implemented), Wordle, Strands, Spelling Bee, Letter Boxed (planned)
- **Two Game Modes**: Story Mode (narrative-driven campaigns) and Game Mode (standalone puzzles)
- **Difficulty Progression**: Easy, Medium, Hard, and unlockable Insane difficulty
- **Story Campaigns**: Multi-scene narratives with character dialogue and plot progression

### VHS Horror Aesthetic
- **Visual Effects**: Authentic VHS tape quality simulation with static, scanlines, and tracking issues
- **Audio System**: Horror ambience, sound effects (click, error, success, jumpscare)
- **Character System**: Video avatars with detective, survivor, reporter, and archivist characters
- **Tape Interface**: Physical VHS tape representations for navigation and selection

### Technical Capabilities
- **Progress Persistence**: LocalStorage-based save system for story completion and unlocks
- **Dynamic Content**: JSON-driven puzzle and story data for easy content expansion
- **Responsive Design**: Canvas-based rendering with adaptive layouts
- **Performance Optimization**: Modular JavaScript architecture with system-based organization

## Target Users
- **Puzzle Game Enthusiasts**: Players who enjoy word games like NYT Connections and Wordle
- **Horror Game Fans**: Users seeking atmospheric horror experiences with puzzle elements
- **Retro Gaming Community**: Players attracted to VHS/80s aesthetic and nostalgic interfaces
- **Content Creators**: Developers wanting to add custom puzzles and stories to the collection

## Value Proposition
Combines the intellectual challenge of popular word puzzles with immersive horror storytelling and authentic retro aesthetics, creating a unique gaming experience that stands apart from traditional puzzle games through its narrative depth and atmospheric presentation.
